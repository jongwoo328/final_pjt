# SCAFY

> SSAFY Cinema Analysis For You

SSAFY 3기 1학기 최종 프로젝트로 제작한 영화 리뷰 커뮤니티 입니다.



## 개요

### 1. 팀명

SCAFY는 SSAFY Cinema Analysis For You의 약자로 사용자 취향으로 영화를 추천해 주는 서비스를 만들고자 하는 팀의 목표가 담긴 이름입니다.
